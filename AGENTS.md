# 作業ルール

## 参照資料

- 変更や修正を行う前に、ルートの `DOCX` フォルダー内にある `掲載原稿.docx` と `制作依頼書.docx` を確認すること。
- 変更内容に関係する範囲で、両方のDOCXを必ず参照すること。
- 掲載原稿に記載のない文言・情報を追加しないこと。追加や変更が必要な場合は、まずユーザーに確認すること。

## TOPページの変更

- `index.html`、TOPページに表示される関連JavaScript、またはTOPページ専用のスタイルを変更してはならない。
- TOPページの変更が必要な場合は、変更内容を先にユーザーへ提示し、ユーザーが明示的に許可した後にのみ変更すること。
- ユーザーの確認・許可がない状態では、TOPページの変更を実施せず、確認待ちとして報告すること。

## 通常の変更手順

1. `DOCX/掲載原稿.docx` と `DOCX/制作依頼書.docx` を確認する。
2. 変更が掲載原稿・制作依頼書に適合するか確認する。
3. TOPページに関係する場合は、ユーザーの明示的な許可を得る。
4. 変更後に差分と必要な検証結果を報告する。
