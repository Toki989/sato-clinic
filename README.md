# 佐藤医院サイト 仕様書

## 1. 概要

本書は、`sato-clinic` の静的Webサイトにおける構成、表示ルール、配色、および更新時の運用方針を定義するものです。

- 対象種別: 医院向け静的Webサイト
- 主ファイル: `index.html`
- スタイル定義: `css/style.css`
- 基本方針: 現在のレイアウトと視覚的な印象を極力維持する

## 2. 対象範囲

本仕様書の対象は以下のとおりです。

- ページ全体の構成
- 各セクションの表示要件
- 見出し表現
- 配色ルール
- 更新時の維持方針

本仕様書では、以下の内容は詳細定義の対象外とします。

- CMS連携
- 動的機能の実装
- バックエンド処理
- 全面的なデザイン刷新

## 3. レイアウト方針

現在の実装を基準レイアウトとします。

- 既存のセクション順は原則として変更しない
- フッターを含む全体の視覚バランスは大きく変更しない
- 余白、カード構造、画像配置は必要がない限り変更しない
- 更新時は既存の `index.html` および `css/style.css` との整合性を優先する

## 4. 文字表現

### 4.1 見出し

- すべての見出しは `Bold` を基本とする

### 4.2 本文

- 本文は可読性を優先し、落ち着いた印象を維持する
- 文字色は基本色の本文色を優先して使用する

## 5. 配色仕様

### 5.1 基本色

サイト全体の印象を決定する主要色は以下のとおりです。

| 用途 | 色コード |
| --- | --- |
| 基本文字色 | `#333333` |
| メインの緑 | `#0A6934` |
| ベースの白 | `#FAFCFB` |
| 強調用オレンジ | `#F59E0B` |
| アクセント用の明るい緑 | `#11B55A` |

### 5.2 補助色

背景、境界線、補助強調、リンク色などに使用する補助色は以下のとおりです。

| 用途 | 色コード |
| --- | --- |
| 補助緑 | `#0F9149` |
| 濃い緑 | `#0B773D` |
| 薄い緑 | `#D8EFDF` |
| ごく薄い緑 | `#E1F6E6` |
| 背景用の淡い緑 | `#E4F8E9` |
| 境界線用の薄緑 | `#C3DEC9` |
| 区切り線用のくすみ緑 | `#6B8F76` |
| 補助強調の緑 | `#18894C` |
| リンク用の緑 | `#2F7A50` |
| 明るい文字色 | `#F3FFF7` |

### 5.3 配色運用ルール

- 主要なUI要素には基本色を優先して使用する
- 補助色は背景、罫線、軽い強弱づけに限定して使用する
- オレンジは重要表示や注意喚起に使用する
- 薄い緑系はやわらかい背景表現に使用する
- 白系の色は清潔感を損なわない範囲で使用する

## 6. ページ構成

ページ構成は以下の順序とします。

1. ヘッダー
2. ヒーローエリア
3. お知らせ / 本日の診療時間
4. 診療内容
5. 写真エリア
6. 診療時間
7. イメージ写真
8. 当院について
9. ご挨拶
10. スタッフ紹介
11. アクセス
12. フッター

## 7. セクション仕様

### 7.1 ヘッダー

- 医院ロゴを表示する
- 電話、アクセス、院内検索、メニューへの導線を配置する
- 現在の横並びユーティリティナビゲーションを維持する

### 7.2 ヒーローエリア

- メインコピーは `地域に根差した医療` とする
- メインビジュアルには医院外観写真を使用する
- 第一印象として清潔感と信頼感を表現する

### 7.3 お知らせ / 本日の診療時間

ヒーローエリア直下に、以下の2つのカードを配置するものとします。

- 病院からのお知らせ
- 本日の診療時間

要件:

- カード形式の表示を維持する
- 情報を短時間で把握できる構成とする
- 現在の左右並びレイアウトを維持する

### 7.4 診療内容

以下の診療項目を表示するものとします。

- 一般内科
- 生活習慣病
- 消化器内科
- 呼吸器内科

要件:

- アイコンを伴う視覚的な案内とする
- 現在のグリッド構成を維持する
- 緑色の枠線を活かした表現を維持する

### 7.5 診療時間

本セクションには以下の情報を含めるものとします。

- 午前診療時間
- 午後診療時間
- 受付時間
- 休診日

現行値:

- 午前: `9:00〜12:00`
- 午後: `13:30〜18:00`
- 受付時間: `8:00〜11:00 / 12:00〜17:00`
- 休診日: `木曜日・土曜午後・日曜日`

### 7.6 当院について

- 医院の理念および地域密着の姿勢を説明する
- テキストと画像のバランスを保った構成とする

### 7.7 ご挨拶

- 院長からの挨拶文を掲載する
- 背景にはやわらかい緑系の面を使用する
- 落ち着きと誠実さのある印象を維持する

### 7.8 スタッフ紹介

以下の職種を掲載対象とします。

- 院長
- 看護師長
- 臨床検査技師
- 医療事務

要件:

- 画像と文章の交互配置を維持する
- 経歴、専門分野、コメント等が読みやすい構成とする

### 7.9 アクセス

本セクションには以下の情報を含めるものとします。

- 住所
- 最寄駅情報
- 駐車場情報
- 来院時の案内文

現行内容:

- 〒123-4567 東京都台東区上野0-0-0
- JR上野駅より徒歩5分
- 無料駐車場10台

### 7.10 フッター

フッターには以下の情報ブロックを含めるものとします。

- 医院情報
- アクセス
- 診療時間
- サイトマップ

要件:

- 現在の多カラム構成を維持する
- 淡い緑を基調とした背景表現を維持する
- 境界線および区切り線の設計を維持する

## 8. デザイントーン

サイト全体を通して、以下の印象を維持するものとします。

- 清潔感
- 信頼感
- 親しみやすさ
- 安心感
- 地域密着感

## 9. 更新時の運用ルール

- コンテンツ更新時は既存レイアウトを崩さないこと
- 色を追加または変更する場合は、基本色と補助色の整合性を確認すること
- 見出し更新時は `Bold` のルールを維持すること
- フッター調整時は既存の背景色、境界線、区切りバランスを優先すること
- 明確な必要がない限り、大きな視覚変更は行わないこと

## 10. 参照ファイル

本仕様書は、以下の実装ファイルを基準として作成しています。

- `index.html`
