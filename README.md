# 佐藤医院 静的Webサイト

佐藤医院の案内サイトです。HTML・CSS・JavaScriptだけで構成された静的サイトで、ビルドツールや外部CMSは使用していません。

## ページ構成・表示順

サイト内のページは、次の順番で案内します。

1. `index.html`: ホーム（トップページ）
2. `01_news.html`: お知らせ
3. `02_services.html`: 診療内容
4. `03_about.html`: 当院について・院長挨拶・スタッフ紹介
5. `04_hours-access.html`: 診療時間・所在地・アクセス
6. `05_faq.html`: よくある質問

フッターのサイトマップと、JavaScriptで生成するヘッダーメニューもこの順番に統一しています。

## ファイル構成

```text
css/style.css              共通スタイルとレスポンシブ対応
js/clinic-hours-data.js    診療時間の唯一のデータソース
js/clinic-hours.js         診療時間表・本日の診療時間・メニューの描画
assets/                    画像・アイコン
```

## 診療時間を変更する場合

診療時間、曜日ごとの診療可否、受付時間を変更するときは、まず `js/clinic-hours-data.js` を修正してください。

このデータを次の場所で共通利用しています。

- トップページの「本日の診療時間」
- トップページの診療時間表
- `04_hours-access.html` の診療時間表
- 各ページのフッター診療時間表
- スマートフォン向け診療時間カード

ページ側に診療時間を直接書き足すと表示が不一致になるため、HTMLへの直書きは避けてください。

## 確認方法

ローカルサーバーで公開する例です。

```powershell
python -m http.server 4173
```

ブラウザで `http://127.0.0.1:4173/` を開き、以下を確認します。

1. トップページの診療内容カードから各診療内容へ移動できること
2. `02_services.html` の各カードがPC・スマートフォンで読めること
3. 診療時間表がトップ・診療時間ページ・フッターで一致すること
4. スマートフォン幅で横スクロールが不要なこと
5. ヘッダーのメニューが開閉し、現在ページが示されること

## 更新時の注意

- 文字コードはUTF-8で保存します。
- 共通スタイルは `css/style.css` にまとめます。
- 画像には内容に応じた `alt` を付け、装飾画像は空の `alt` とします。
- 診療時間などの医療情報を更新した場合は、関連するお知らせの内容も確認します。
